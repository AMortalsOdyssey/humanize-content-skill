## 适用场景

只在用户明确要求以下内容时参考：

- `shuorenhua`。
- “说人话”。
- 中文优先去 AI 味。
- 保事实、分场景、改完可直接发。
- 不想要“每个字都对，但一看就不是人写的”中文。

适合处理：

- 日常聊天。
- 技术状态同步。
- README / release note。
- 论坛帖、issue 回复。
- 中文长文、公开介绍、产品说明。
- 需要保护数字、版本、命令、路径、报错、事实归属和引用证据的文本。

不要用于：

- 需要英文 Wikipedia AI-writing 模式清理的文本；这类优先看 `humanizer`。
- 需要强个人风格模仿的长文；这类优先看 `khazix-writer` 或用户点名的风格规则。
- 需要蒸馏公众人物心智模型和表达 DNA 的任务；这类优先看 `nuwa-skill`。
- 视频脚本或图片提示词。

# 文案规则引用：Humanizer-zh

这是文案类规则下的一个外部引用，不是本仓库内置规则，也不是默认文风。

## 来源

- 仓库：https://github.com/op7418/Humanizer-zh
- 规则文件：https://github.com/op7418/Humanizer-zh/blob/main/SKILL.md
- 英文原版：https://github.com/blader/humanizer

## 适用场景

只在用户明确要求以下内容时参考：

- `Humanizer-zh`。
- 中文版 Humanizer。
- 中文 AI 写作去痕。
- 想按 blader/humanizer 的中文翻译版清理文本。
- 需要对照常见 AI 写作痕迹做中文改写。

适合处理：

- 营销文案里的夸张形容词。
- 中文博客、摘要、介绍里的 AI 腔。
- “此外 / 至关重要 / 深入探讨 / 证明 / 格局”等模板词。
- 过度完整、过度平滑、缺少真实细节的 AI 生成段落。

不要用于：

- 用户要求某个具体作者的个人风格。
- 需要保留强烈平台语感的小红书原生文案。
- 视频脚本。
- 图片提示词。
- 未点名该规则且需要深度方法论结构的文章。

## 使用边界

本仓库不 vendoring 这个外部仓库，不保存其完整内容。需要精确使用时，访问上面的 GitHub 链接读取原规则；只需要知道它存在时，把它作为文案类规则列表中的一个外部参考即可。

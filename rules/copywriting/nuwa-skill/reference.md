## 适用场景

只在用户明确要求以下内容时参考：

- `nuwa-skill`。
- “女娲 skill”。
- 蒸馏某个人、作者、博主或专家的写作 / 思考方式。
- 提取表达 DNA、语气节奏、用词偏好。
- 提取心智模型、认知框架、决策启发式和反模式。
- 想把某个公众人物或主题方法论做成可复用的写作视角。

适合处理：

- 从公开文章、访谈、播客、社媒中总结一个人的表达方式。
- 给某个作者风格建立可复用规则。
- 需要同时保留“怎么说话”和“怎么判断”的文案任务。
- 创建新的个人风格 skill 或主题写作 skill。

不要用于：

- 普通去 AI 味改写。
- 没有足够样本材料的个人风格模仿。
- 要求虚构某人真实想法的内容。
- 需要严格事实保真的新闻、法律、医疗、财务文本。

## 使用边界

Nuwa 的价值在于蒸馏“公开材料中可观察到的表达和判断方式”，不是复刻一个真实的人。使用时必须保留诚实边界：公开表达不等于真实想法；材料不足时，只能输出初步风格假设。

本仓库不 vendoring 这个外部仓库，不保存其完整内容。需要精确使用时，访问上面的 GitHub 链接读取原规则；只需要知道它存在时，把它作为文案类规则列表中的一个外部参考即可。
